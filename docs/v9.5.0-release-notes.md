# v9.5.0

## Chandra Darshana Visibility and Satsangi Jeevan Observance Expansion

This release brings better support for observing **Chandra Darshana** — the first visible waxing crescent of the moon after the new moon. It also adds several meaningful new festivals and vrats from the Swaminarayan tradition, improves fasting guidance for Ekadashi, and includes clearer documentation about how these calculations are made.

All English, Hindi, and Gujarati data files for 2026 (both Amanta and Purnimanta calendars) have been regenerated with the latest rules.

### Highlights

**Chandra Darshana (First Visible Crescent)**  
The package now calculates Chandra Darshana observances more accurately. It looks for the first likely visible crescent after Amavasya, typically on Shukla Pratipada or Dwitiya.  

For dates where it applies, the output now includes easy-to-use timing information:
- A clear visibility window (when the crescent is expected to be visible after sunset)
- Start and end times in both readable format and precise timestamps
- Duration of the visibility period
- Details on how visibility was assessed (moon position, brightness, time since sunset, etc.)

This makes it much easier to show users when to look for the moon or plan the observance.

**More Festivals and Vrats**  
The total has grown to **329 unique festivals** and **90 unique vrats**. Many new additions come from the Swaminarayan and related traditions:

- Ramanand Swami Appearance Festival
- Nara-Narayan Arjun Janmotsav (including Arjun Janmotsav)
- Chandrayan Vrat
- Dharmadev Janmotsav
- Hatadi Festival
- Swaminarayan Kurma Jayanti
- Swaminarayan Snanyatra
- Swaminarayan Rathyatra
- Hindola Festival Begins
- Pavitra Festival
- Swaminarayan Varaha Jayanti
- Dhanatrayodashi (Dhanteras) and Alankar Marjan
- Danleela Mahotsav, Padma Ekadashi, Gangavatar, Dasahara, and other refinements

**Improved Ekadashi Fasting Guidance**  
Ekadashi entries can now include helpful fasting guidance based on the Satsangi Jeevan tradition. This includes general guidance, allowances for those who may not be able to fast strictly, and special stricter guidance for Prabodhini Ekadashi. The guidance is available in English, Hindi, and Gujarati.

**Clearer Explanation of Rules**  
A new dedicated document explains exactly how Chandra Darshana is decided — separating traditional Panchang practice from the modern visibility checks used for practical prediction. Traditional source references have also been expanded across the documentation.

### Documentation Improvements

- New document: **Chandra Darshana Decision Tree** — explains the logic behind the observance in clear terms
- Major updates to the **Traditional Text Sources** document with more details on the sources used for festivals, Ekadashi, eclipses, and regional practices
- Updated counts and descriptions in the main documentation and README

### Regenerated Data Files (2026)

All output files for 2026 have been refreshed for both calendar systems (Amanta and Purnimanta) and all supported languages (English, Hindi, Gujarati):

- Daily “today” data
- June 2026 month calendar
- Full festival lists and festival-only files
- Vrat lists
- Eclipse data (2026–2032)
- Raw detailed output

### Upgrade Notes

- **Regenerate your data**: If you store or cache any of the generated JSON files (today, month, festivals, vrats, etc.), please regenerate them after updating. The files now contain the new festivals, Chandra Darshana timings, and other improvements.
- **New information available** (optional to use):
  - Chandra Darshana entries now provide a `visibility_window` and `observance_window` with readable times and duration.
  - Ekadashi entries may include a `fasting_guidance` section with practical advice.
- These new fields are additive. Existing code that reads the data will continue to work without changes.
- The visibility calculations for Chandra Darshana are based on practical astronomical checks and are clearly marked as such in the documentation (not claimed as direct classical rules).
